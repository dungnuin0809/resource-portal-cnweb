# Resource Portal CNWeb

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涉及各类通用资源、工具或信息展示，不针对任何特定网站或域名，旨在提供一个结构化的静态页面集合。

仓库本身不包含后端逻辑或动态内容，所有页面均为独立的 HTML 文件，可直接通过浏览器打开或部署到静态托管服务。

## 目录说明

```
.
├── pages/          # HTML 页面存放目录
├── assets/         # 共用资源文件（样式、脚本、图片等）
├── index.html      # 入口页面或导航页
└── README.md       # 本文件
```

- **pages/**：放置所有独立 HTML 页面，按功能或主题命名子目录。
- **assets/**：存放页面共享的 CSS、JavaScript、图片等资源，避免重复引用。
- **index.html**：提供页面索引或跳转功能，方便浏览者快速定位。

## 页面归档说明

- 每个页面均为独立自包含的 HTML 文件，部分页面可能引用 `assets/` 下的资源。
- 页面命名应简洁明了，反映其内容或用途。
- 欢迎通过 Pull Request 提交新的页面或改进现有页面，但请确保内容不涉及具体商业推广或域名绑定。
- 所有页面应保持基本可访问性，避免依赖外部不可控资源。
